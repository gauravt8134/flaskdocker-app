🌐 Flask Web App — Dockerized & Deployed on AWS EC2

A lightweight web application built with Flask and served through a Docker container, deployed on an AWS EC2 instance.
🚀 Tech Stack

Python / Flask — Web framework powering the UI and backend logic

Docker — Containerized for consistent, portable deployment

AWS EC2 — Cloud server hosting the live application


⚙️ How It Works :

The Flask app is packaged into a Docker image

The image runs as a container on an EC2 instance

The app is accessible via the EC2 public IP / domain
